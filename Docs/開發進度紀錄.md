## 二、最新 Build 修改內容

詳見 `Changelog.md`。以下為 Build 14 重點摘要：

- `FeedbackView` 兩步驟改版，附截圖上傳、系統資訊自動帶入、email 選填
- 新增 `MyFeedbackView` 歷史回饋列表
- `Expense.receiptImagePathsData` 支援多張收據照片，新增 `ReceiptPhotoManager`
- `AddExpenseView`、`EditExpenseView` 加入拍照/相簿收據入口
- 新增 `CurrencyPicker` 元件，幣種選單五處改為 Menu + 更多幣種
- 統計頁改版：累計統計 + 年度 Picker 切換 + 可收合幣種卡片
- `DietaryParser` 補充不吃辣、糖尿病飲食、低鈉飲食；AI badge 加注；UserDefaults 快取
- Feature flag 機制建立（`AppConfigManager`）
- `CameraView` 抽至 `Components/`；`LibraryPickerView` 移除
- Debug 工具：`DebugDataGenerator` 產生/清除測試資料

### 4/28 內部調整

- 檔案結構整理：拼字修正（`JournalDetailVIew` / `HotelSuporting`）、`Views/Member/` 收 `TourMember*`、`Utilities/` 收 `SeedData` / `DebugDataGenerator`、`Docs/`（repo 根）收 markdown 文件、`PrivacyInfo.xcprivacy` 移專案根、`SupabaseManager` extension 與 main 並列於 `Managers/`
- Debug build 強制啟用所有 feature flag（`#if DEBUG` 短路）
- `DebugDataGenerator` 用 `context.transaction` 修 SwiftData crash；移除無作用的 `Task.detached + MainActor.run` wrapping；`testMemberCount` 常數化避免 paxCount 不一致
- 死碼清除：移除未使用的 `CurrencyPicker.CurrencyInfo` struct

## 四、Roadmap

### Phase 6：登入系統與使用者體驗改善（上架前）

- ✅ 意見回饋改版（兩步驟流程、附截圖最多3張、系統資訊自動帶入、email 選填）
- ✅ 我的回饋列表（device_id 查詢歷史回饋、status / developer_reply 顯示）
- ✅ 幣種選單擴充（CurrencyPicker 元件、分區顯示、常用/最近使用排前）
- ✅ 收據拍照存檔（多張、拍照/相簿、ReceiptPhotoManager）
- 登入系統（Apple ID / Google）
- Landing Page（首次開啟顯示、登入後不再出現）

### Phase 7：上架準備（上架前）

- Supabase RLS
- 地點資料歷史版本紀錄
- App Store 正式上架

### Phase 8：上架後基礎建設

- 更新 App ID 評分連結
- App 分享功能
- Vercel 基礎建設（第三方 API key 保護、隱私權宣告頁、行銷頁、說明手冊）
- App 內功能說明頁

### Phase 9：認證領隊功能

- 認證領隊權限分層
- 奧客黑名單
- 景點廁所及細節資料欄位擴充
- 貢獻度基礎建設

### Phase 10：進階功能

- 收據 OCR 解析（Vision + Apple Intelligence）
- 日誌地點選取（飯店單選、餐廳單選、景點多標記）
- 貢獻度排行榜與徽章

### Phase 11：航班查詢

- 航班資訊查詢（Vercel proxy + AviationStack，認證領隊限定）
